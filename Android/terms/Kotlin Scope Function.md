# Kotlin Scope Function
### apply
생성과 동시에 초기화하고 자기자신을 return

### also
자기 자신이 필요한데, 초기화를 좀 더 쉽게(수신 객체를 사용하지 않거나, 수신 객체의 속성을 변경하지 않고 사용할 때)

### with
(생성과 동시에 초기화, null이 될 수 없는) 결과가 필요하지 않을 경우

### run
객체의 값의 접근을 쉽게 할 때

### let
객체의 값이 명확해야 할 때
